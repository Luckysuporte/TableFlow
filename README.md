# TableFlow

TableFlow is a modern, futuristic financial control application for proprietary trading.

## Prerequisites

- Node.js (v16 or higher)
- npm (v8 or higher)

## Installation

1. Open a terminal in this directory.
2. Install dependencies:
   ```bash
   npm install
   ```

## Running the App

1. Start the development server:
   ```bash
   npm run dev
   ```
2. Open your browser and navigate to the URL shown in the terminal (usually `http://localhost:5173`).

## Features

- **Authentication**: Create account and login (local storage based).
- **Dashboard**: Overview of your trading goals.
- **Goal Tracker**: Set and track your financial goals with visual progress.
- **Daily Log**: Record your daily trading results (Win/Loss).
- **Withdrawals**: Calculate and track your withdrawals with retention rates.
- **Responsive Design**: Works on Desktop, Tablet, and Mobile.
- **Dark Theme**: Elegant dark mode with purple gradients.

## Technologies

- React
- Vite
- Framer Motion (Animations)
- Lucide React (Icons)
- CSS Variables (Theming)
